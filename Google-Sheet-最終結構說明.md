# 📊 Google Sheet 最終結構說明

## ✅ 只需要 3 個頁簽！

---

## 📋 Google Sheet 結構總覽

你的 Google Sheet 應該包含以下 **3 個頁簽**：

```
Google Sheet: AI-Global (或任何你喜歡的名字)
├── 頁簽1: 推廣人員
├── 頁簽2: Global
└── 頁簽3: 評估地點
```

---

## 📑 頁簽 1：推廣人員

### **用途：**
推廣代碼與推廣人員郵箱的對照表

### **表格結構：**

```
     A            B                C
┌──────────┬────────────────┬──────────┐
│ 推廣代碼 │      郵箱       │   姓名   │
├──────────┼────────────────┼──────────┤
│   001    │ user1@gmail.com│  張三    │
│   002    │ user2@gmail.com│  李四    │
│   003    │ user3@gmail.com│  王五    │
└──────────┴────────────────┴──────────┘
```

### **欄位說明：**
- **第A列 - 推廣代碼**：唯一識別碼（例如：001, 002, A01, TW001 等）
- **第B列 - 郵箱**：推廣人員的聯絡郵箱（必填）
- **第C列 - 姓名**：推廣人員姓名（選填，方便辨識）

### **範例數據（可直接複製）：**
```
推廣代碼	郵箱	姓名
001	promoter1@gmail.com	張三
002	promoter2@gmail.com	李四
003	promoter3@gmail.com	王五
TW01	taiwan.agent@gmail.com	陳小明
MY01	malaysia.agent@gmail.com	Lee Wei
```

## 📑 頁簽 2：Global

### **用途：**
儲存客戶報名的資料

### **表格結構：**

```
   A         B       C        D        E       F       G        H          I           J
┌────────┬────────┬────────┬────────┬────────┬────────┬────────┬────────┬──────────┬──────────┐
│時間戳記│  姓名  │電子郵件│電話號碼│國家地區│  行業  │評估地區│LINE ID │WhatsApp  │推廣代碼  │
├────────┼────────┼────────┼────────┼────────┼────────┼────────┼────────┼──────────┼──────────┤
│ 自動   │ 客戶名 │  Email │ Phone  │ TW/MY  │ 科技業 │ 台北...│ line123│ +886...  │   001    │
└────────┴────────┴────────┴────────┴────────┴────────┴────────┴────────┴──────────┴──────────┘
```

### **欄位說明：**
- **第A列 - 時間戳記**：系統自動填入提交時間
- **第B列 - 姓名**：客戶姓名
- **第C列 - 電子郵件**：客戶郵箱
- **第D列 - 電話號碼**：客戶電話
- **第E列 - 國家地區**：客戶選擇的國家（TW 或 MY）
- **第F列 - 行業**：客戶所屬行業
- **第G列 - 評估地區**：客戶選擇的活動場次
- **第H列 - LINE ID**：客戶的 LINE ID
- **第I列 - WhatsApp號碼**：客戶的 WhatsApp 號碼
- **第J列 - 推廣代碼**：客戶使用的推廣代碼

### **設定標題行（第一行）：**
```
時間戳記	姓名	電子郵件	電話號碼	國家地區	行業	評估地區	LINE ID	WhatsApp號碼	推廣代碼
```

⚠️ **注意：** 這個頁簽的標題行（第一行）必須手動創建，第二行開始的數據會由系統自動填入。

## 📑 頁簽 3：評估地點

### **用途：**
管理不同國家的活動場次選項

### **表格結構：**

```
     A        B           C                                    D
┌─────────┬────────┬──────────────────────────────┬──────────────┐
│   ID    │  國家  │         地點描述              │   是否啟用   │
├─────────┼────────┼──────────────────────────────┼──────────────┤
│   tw1   │   TW   │ 11/25 星期一 晚上7:00...     │     是       │
│   tw2   │   TW   │ 11/27 星期三 下午2:00...     │     是       │
│   my1   │   MY   │ 12/1 Monday 7pm KL...        │     是       │
└─────────┴────────┴──────────────────────────────┴──────────────┘
```

### **欄位說明：**
- **第A列 - ID**：唯一識別碼（例如：tw1, tw2, my1）
- **第B列 - 國家**：國家代碼，**必須填 TW 或 MY**（大寫）
- **第C列 - 地點描述**：完整的活動信息（日期、時間、地點）
- **第D列 - 是否啟用**：填 `是` 顯示，填 `否` 隱藏（留空也視為啟用）

### **範例數據（可直接複製）：**

#### **台灣的活動：**
```
ID	國家	地點描述	是否啟用
tw1	TW	11/25 星期一 晚上 7:00~9:00 台北市信義區忠孝東路五段123號	是
tw2	TW	11/27 星期三 下午 2:00~4:00 台中市西屯區台灣大道四段888號	是
tw3	TW	11/29 星期五 晚上 7:00~9:00 高雄市前鎮區中山四路100號	是
```

#### **馬來西亞的活動：**
```
my1	MY	12/1 Monday 7:00pm~9:00pm Kuala Lumpur, Jalan Ampang, Empire City Mall	是
my2	MY	12/3 Wednesday 2:00pm~4:00pm Kuala Lumpur, KLCC Convention Centre	是
my3	MY	12/5 Friday 7:00pm~9:00pm Petaling Jaya, Paradigm Mall	是
```

### **💡 調整顯示順序：**
直接在 Google Sheet 中**拖動行**即可！不需要填數字。
- 選中行號（點擊左側數字）
- 拖動到想要的位置
- 系統會按照 Sheet 中的行順序顯示

## 📝 完整設置步驟

### **步驟 1：創建 Google Sheet**
1. 前往 https://sheets.google.com
2. 點擊「空白」創建新的 Sheet
3. 將 Sheet 命名為：`AI-Global`（或任何你喜歡的名字）

### **步驟 2：創建頁簽**
1. 將預設的「工作表1」重新命名為：**推廣人員**
2. 點擊左下角「+」新增頁簽，命名為：**Global**
3. 再次點擊「+」新增頁簽，命名為：**評估地點**

### **步驟 3：設置「推廣人員」頁簽**
1. 切換到「推廣人員」頁簽
2. 在第一行輸入標題：
   ```
   推廣代碼	郵箱	姓名
   ```
3. 從第二行開始填入推廣人員資料

### **步驟 4：設置「Global」頁簽**
1. 切換到「Global」頁簽
2. 在第一行輸入標題：
   ```
   時間戳記	姓名	電子郵件	電話號碼	國家地區	行業	評估地區	LINE ID	WhatsApp號碼	推廣代碼
   ```
3. **不要填入任何數據**（第二行開始留空，系統會自動填入）

### **步驟 5：設置「評估地點」頁簽**
1. 切換到「評估地點」頁簽
2. 在第一行輸入標題：
   ```
   ID	國家	地點描述	是否啟用
   ```
3. 從第二行開始填入活動場次資料

### **步驟 6：複製 Sheet ID**
1. 複製瀏覽器地址欄中的 Sheet ID：
   ```
   https://docs.google.com/spreadsheets/d/【這裡就是ID】/edit
   ```
2. 保存這個 ID，稍後需要貼到 Apps Script 中

## 🎯 重要提醒

### ✅ 必須遵守的規則：
1. **頁簽名稱**必須完全一致：
   - 推廣人員（不是「推广人员」或其他）
   - Global（不是「global」或「GLOBAL」）
   - 評估地點（不是「评估地点」或其他）

2. **標題行**（第一行）必須手動創建

3. **國家代碼**必須是大寫的 `TW` 或 `MY`

4. **是否啟用**欄位填 `是` 或 `否`（留空也可以）

### ⚠️ 頁簽順序：
雖然我們建議的順序是：
1. 推廣人員
2. Global
3. 評估地點

但實際上**頁簽順序不影響**系統運作！系統是通過**頁簽名稱**來讀取數據的，不是通過順序。

所以你可以自由調整頁簽順序，只要名稱正確就可以了！😊

## 📞 常見問題

### Q1: 可以改頁簽名稱嗎？
**A:** 如果要改，必須同步修改 Google Apps Script 中的代碼。不建議更改。

### Q2: 可以添加更多欄位嗎？
**A:** 可以在最右邊添加額外欄位（備註、標籤等），不會影響系統運作。但**不要**刪除或移動現有欄位。

### Q3: 評估地點的順序如何調整？
**A:** 直接在 Google Sheet 中拖動行即可，系統會按照 Sheet 中的行順序顯示地點。

### Q4: 國家代碼可以小寫嗎？
**A:** 不可以，必須是大寫的 `TW` 或 `MY`。

### Q5: Global 頁簽需要手動填數據嗎？
**A:** 不需要！只需要設置標題行（第一行），第二行開始的數據會由系統自動填入。
